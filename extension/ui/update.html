Native messenger update

Why am I getting this?

The installed Pywalfox native messaging application is outdated.
If you do not fix this, you may experience issues and certain features may not be available.

If you encounter bugs with the new version, refer to the open issues on GitHub (or create a new one).

GitHub issues Installation guide Troubleshooting guide

If you have previously installed the Pywalfox native messaging application, you can simply run the following command to update to the latest version (assuming you installed it using pip):

pip install --upgrade pywalfox
